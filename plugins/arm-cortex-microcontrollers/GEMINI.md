# Arm Cortex Microcontrollers

## Agents

In Gemini CLI, invoke agents by describing your task naturally. The model activates the appropriate expertise.

| Agent | Model | When to use |
|---|---|---|
| `arm-cortex-expert` | inherit | > Senior embedded software engineer specializing in firmware and driver development for ARM Cortex-M microcontrollers... |

**Example natural language triggers:**

- "> Senior embedded software engineer specializing in firmware and driver development for ARM Cortex-M microcontrollers (Teensy, STM32, nRF52, SAMD)" → activates `arm-cortex-expert`
