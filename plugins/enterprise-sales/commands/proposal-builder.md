# Commercial Proposal Builder

You will create a professional, executive-ready commercial proposal for VK Cloud solutions.

## Process

### 1. Gather Requirements

**Customer Information**:
- Company name, industry, size
- Key stakeholders (who will read this proposal?)
- Pain points and business objectives
- Current technology environment

**Solution Scope**:
- Which VK Cloud products/services?
  - VK Public Cloud (compute, storage, networking)
  - VK Private Cloud
  - VK Kubernetes
  - VK Data Platform (databases)
  - VK Object Storage
  - Professional services (implementation, training)
  - Support level (standard, premium, enterprise)

**Technical Requirements**:
- Compute: # of VMs, vCPU, RAM, GPU
- Storage: Capacity (TB), IOPS requirements
- Kubernetes: # of clusters, nodes, workloads
- Databases: Type (PostgreSQL, ClickHouse, Redis, Kafka), size, connections
- Object Storage: Capacity, tier (hot, cold, archive)
- Networking: Bandwidth, VPN/Direct Connect

**Commercial Parameters**:
- Budget range
- Contract term (1, 2, 3 years)
- Payment terms preference (monthly, quarterly, annual prepay)
- Implementation timeline

### 2. Generate Proposal

Create a structured proposal document with the following sections:

#### Executive Summary (1 page)
- Customer's business challenge
- Proposed solution overview
- Key benefits and business outcomes
- Investment summary
- Recommended next steps

#### Business Case (2-3 pages)
**Current State Challenges**:
- Pain points with quantified impact
- Cost of inaction
- Risks of status quo

**Proposed Solution**:
- Architecture overview (high-level diagram)
- How solution addresses each pain point
- Key capabilities and features
- Differentiation vs. alternatives

**Business Value**:
- Quantified benefits (ROI, cost savings, revenue impact)
- Risk mitigation and compliance benefits
- Strategic advantages

#### Technical Solution (2-4 pages)
**Architecture**:
- High-level architecture diagram
- Component descriptions
- Integration points
- Scalability and high availability design

**Bill of Materials**:
| Component | Specification | Quantity | Unit Price | Total |
|-----------|--------------|----------|------------|-------|
| VK Kubernetes Cluster | 3 nodes, 8 vCPU, 32GB RAM | 1 | $XXX/month | $XXX |
| VK Data Platform (PostgreSQL) | 50GB storage, HA | 1 | $XXX/month | $XXX |
| VK Object Storage | 10TB, Hot tier | 10TB | $XX/GB/month | $XXX |
| Professional Services | Implementation | 40 hours | $XXX/hour | $XXX |
| **Monthly Total** | | | | **$X,XXX** |
| **Annual Total (12 months)** | | | | **$XX,XXX** |

#### Pricing & Terms (1-2 pages)
**Pricing Options**:

**Option 1 - Annual Prepay (Recommended)**:
- Total: $XX,XXX/year (15% discount)
- Payment: Annual prepayment
- Savings: $X,XXX vs. monthly

**Option 2 - Quarterly Prepay**:
- Total: $XX,XXX/year (5% discount)
- Payment: Quarterly in advance
- Savings: $X,XXX vs. monthly

**Option 3 - Monthly**:
- Total: $XX,XXX/year
- Payment: Monthly arrears
- Flexibility: Pay-as-you-go

**Multi-Year Discounts**:
- 2-year: 10% additional discount
- 3-year: 15% additional discount

**Contract Terms**:
- Contract duration: [1/2/3] years
- Auto-renewal: [Yes/No, with X-day notice period]
- Payment terms: [Net 30/60/90]
- SLA: 99.95% uptime guarantee
- Support: [Standard/Premium/Enterprise]

#### Implementation Plan (1 page)
**Phase 1 - Planning (Weeks 1-2)**:
- Requirements finalization
- Architecture design review
- Resource allocation

**Phase 2 - Implementation (Weeks 3-6)**:
- Infrastructure provisioning
- Application migration
- Integration configuration

**Phase 3 - Testing (Weeks 7-8)**:
- Functional testing
- Performance validation
- Security audit

**Phase 4 - Go-Live (Week 9)**:
- Production cutover
- User training
- Hypercare support

**Phase 5 - Optimization (Weeks 10-12)**:
- Performance tuning
- Cost optimization
- Knowledge transfer

#### Why VK Cloud? (1 page)
**Competitive Advantages**:
- Data sovereignty (152-FZ compliance)
- 30% cost savings vs. AWS/Azure
- No sanctions risk
- Russian-language support
- Local infrastructure and partnerships

**Customer Proof Points**:
- [Similar customer] achieved [result]
- [Industry reference] migrated from [competitor] and saved $X
- [Case study link]

**About VK Cloud**:
- Part of VK Group (largest Russian tech company)
- [X] years operating internet-scale infrastructure
- [X] enterprise customers
- GOST certifications and compliance

#### Next Steps
1. Proposal review meeting: [Date]
2. Q&A and technical deep dive: [Date]
3. POC (if required): [Dates]
4. Contract negotiation: [Dates]
5. Target signature date: [Date]
6. Implementation kickoff: [Date]

#### Appendix
- Detailed architecture diagrams
- SLA definitions
- Security and compliance certifications
- Terms and conditions
- Customer references (with permission)

### 3. Customize for Audience

**For CFO**:
- Lead with financial impact (ROI, TCO, payback)
- Emphasize cost predictability and risk mitigation
- Include sensitivity analysis

**For CIO/CTO**:
- Lead with technology strategy and innovation
- Emphasize architecture, scalability, security
- Include technical specifications

**For CEO**:
- Lead with strategic value and competitive advantage
- Emphasize business outcomes and market position
- Keep it concise (10 pages max)

**For Procurement**:
- Lead with pricing transparency and benchmarks
- Emphasize contract terms and compliance
- Include payment options

## Output Format

Provide proposal in clean, professional markdown format that can be:
- Converted to PDF for distribution
- Presented in executive meetings
- Shared with procurement and legal teams

Use:
- Clear section headings
- Tables for pricing and specifications
- Bullet points for readability
- Bold for emphasis
- Professional, consultative tone

## Tips

- **Executive-Level Language**: Business outcomes, not technical jargon
- **Quantify Everything**: Use numbers, not adjectives
- **Visual Aids**: Include diagrams, charts, tables
- **Proof Points**: Customer references, case studies, benchmarks
- **Clear Pricing**: No hidden costs, transparent breakdown
- **Action-Oriented**: Specific next steps with dates
- **Professional Format**: Clean, consistent, branded
- **Right-Sized**: Match detail level to audience (10-15 pages for executives)
