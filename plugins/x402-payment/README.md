# x402 Payment Plugin

Monetize any MCP server with zero code changes. Pay-per-call billing via HTTP 402 protocol.

## Features
- One-line integration with existing MCP servers
- Stripe fiat payments + USDC crypto payments
- Automatic payment verification via webhooks
- 19.8% affiliate revenue sharing
- No subscriptions, no monthly fees — pure pay-per-call

## Installation
```bash
npx skills add https://github.com/scotia1973-bot/scotia1973-bot-agents
/plugin install x402-payment
```
