# Context Management

## Agents

In Gemini CLI, invoke agents by describing your task naturally. The model activates the appropriate expertise.

| Agent | Model | When to use |
|---|---|---|
| `context-manager` | inherit | Elite AI context engineering specialist mastering dynamic context management, vector databases, knowledge graphs, and... |

## Commands

These commands are available in Claude Code via slash command. In Gemini CLI, describe the equivalent task in natural language.

| Claude Code Command | Description |
|---|---|
| `/context-management:context-restore` | Context Restoration: Advanced Semantic Memory Rehydration |
| `/context-management:context-save` | Context Save Tool: Intelligent Context Management Specialist |

## Gemini CLI Usage

**Example natural language triggers:**

- "Elite AI context engineering specialist mastering dynamic context management, vector databases, knowledge graphs, and intelligent memory systems" → activates `context-manager`
- In Claude Code: `/context-management:context-restore` — in Gemini: describe the task in natural language
