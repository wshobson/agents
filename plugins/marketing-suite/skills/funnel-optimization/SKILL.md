# Funnel Optimization

## Когда использовать

- Анализ conversion funnel и выявление bottlenecks
- Оптимизация signup/onboarding flows
- Improving trial-to-paid conversion
- Reducing friction в checkout process
- Multi-step form optimization

## Ключевые концепции

### Funnel Analysis Framework

**Standard SaaS Funnel:**
```
Visitor → Signup → Activated → Retained → Paid → Advocate
 100%      40%       60%        70%       30%     20%

Drop-offs:
- 60% never signup (awareness/value prop issue)
- 40% signup but don't activate (onboarding problem)
- 30% activate but don't retain (product-market fit)
- 70% retained but don't pay (pricing/value perception)
```

### Optimization Techniques

**1. Reduce Friction**
- Minimize form fields (each field = 5-10% drop)
- Social login options
- Progressive disclosure (ask for info when needed)
- Clear error messages
- Auto-fill and smart defaults

**2. Increase Motivation**
- Social proof (testimonials, logos, user counts)
- Urgency/scarcity (limited time offers)
- Value preview (show before ask)
- Trust signals (security badges, guarantees)

**3. Optimize Messaging**
- Benefit-focused headlines
- Clear CTAs (action-oriented)
- Microcopy optimization
- Progress indicators

## Best Practices от Leaders

**Amazon 1-Click**: Reduce friction to minimum (patent!)

**Netflix Onboarding**:
- Email → Password → Payment → Done (3 steps)
- Start watching immediately after signup
- No email confirmation required

**Shopify Free Trial**:
- No credit card upfront
- 14 days full access
- Conversion happens after seeing value

**Stripe Checkout**:
- One-page checkout
- Mobile-optimized
- Apple Pay / Google Pay integration
- 50% faster vs traditional forms
