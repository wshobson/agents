# Retention Tactics

## Когда использовать

- Retention rate declining (need intervention)
- Building habit-forming product features
- Designing reactivation campaigns
- Creating engagement loops
- Reducing early churn

## Ключевые концепции

### Hook Model (Nir Eyal)

```
Trigger → Action → Variable Reward → Investment
   ↑                                      ↓
   └──────────────────────────────────────┘
```

**Examples:**
- **Instagram**: Notification (trigger) → Open app (action) → See likes (reward) → Post photo (investment)
- **LinkedIn**: Email (trigger) → Visit profile (action) → Connection request (reward) → Update profile (investment)

### Retention Curve Types

**Good Retention** (flattening curve):
```
Day 0:  100%
Day 1:  40%
Day 7:  25%
Day 30: 20% ← Flattens = found product-market fit
```

**Bad Retention** (declining to zero):
```
Day 0:  100%
Day 1:  30%
Day 7:  10%
Day 30: 2% ← Continues declining = no PMF
```

### Reactivation Strategies

**Email Sequences:**
```
Day 7 (inactive): "We miss you" + What's new
Day 14: Customer story + Use case reminder
Day 21: Special offer + Incentive
Day 30: Survey "Why did you leave?"
```

**In-App Triggers:**
- Push notifications (personalized, not spammy)
- In-product messages (feature discovery)
- Progress indicators (LinkedIn profile strength)
- Streaks & achievements (Duolingo)

## Best Practices

**Spotify Retention**:
- Personalized playlists (Discover Weekly)
- Daily Mix (habit formation)
- Offline mode (reduce friction)
- Social sharing (network effects)

**Slack Retention**:
- 2000 messages = magic number
- Team-based (network effect)
- Integrations (sticky)
- Search history (accumulated value)

**Duolingo**:
- Daily streaks (loss aversion)
- Push notifications (smart timing)
- Gamification (XP, levels)
- Social competition (leaderboards)
