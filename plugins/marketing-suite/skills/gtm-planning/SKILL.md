### Launch Tiering Framework (AWS Model)

**Tier 1: Transformational Launch**
```
Scope: New product, platform, –∏–ª–∏ game-changing capability
Frequency: 1-2 per year
Investment: $500K - $2M+
Timeline: 12+ weeks planning
Channels: All channels activated

Examples:
- AWS Lambda launch (serverless computing)
- Stripe Checkout (new product line)
- OpenAI ChatGPT launch

Components:
- Major analyst event (AWS re:Invent style)
- Press coverage (tier-1 media)
- Executive keynotes
- Customer advisory preview
- Partner ecosystem activation
- Global field marketing
- Multi-million paid media
```

**Tier 2: High-Impact Launch**
```
Scope: Major feature or significant capability
Frequency: Quarterly
Investment: $50K - $200K
Timeline: 6-8 weeks
Channels: Digital + select field

Examples:
- New integration (Salesforce + Slack)
- Major AI/ML capability
- Enterprise feature set

Components:
- Blog post + press release
- Webinar series
- Email campaigns
- Sales enablement
- Partner co-marketing
- Targeted paid media
- Customer spotlights
```

**Tier 3: Continuous Deployment**
```
Scope: Minor enhancement or improvement
Frequency: Weekly/bi-weekly
Investment: $5K - $20K
Timeline: 1-2 weeks
Channels: Owned channels only

Examples:
- UI improvements
- API updates
- Performance enhancements

Components:
- Release notes
- In-app notifications
- Email to active users
- Documentation updates
- Sales knowledge base
```

## Success Metrics Framework

```

- Social reach: 1M+ impressions
- Website traffic: +50% vs baseline
- Brand search volume: +100%

### Consideration Tier (Middle of Funnel)
```
Target Metrics (First 90 Days):
- Content downloads: 1,000+
- Webinar registrations: 500+
- Demo requests: 200+
- Free trial signups: 100+
- Sales meetings: 50+ qualified
```

### Conversion Tier (Bottom of Funnel)
```
Target Metrics (First Quarter):
- MQLs generated: 500+
- SQLs converted: 100+ (20% conversion)
- Opportunities: $2M+ pipeline
- Closed-won: $500K+ ARR

```

- Expansion revenue: 20% of base

## Channel-Specific Playbooks

### Product-Led Growth (PLG) Playbook

**Framework (Stripe/Notion Model):**
```
Step 1: Frictionless Signup
- No credit card required
- Social login (Google/GitHub)
- 2-minute time to first value

Step 2: Aha Moment Acceleration
- Interactive onboarding (not video)
- Progressive disclosure (show when needed)
- Quick win prompts

Step 3: Usage-Based Expansion
- Free tier ‚Üí Paid tier triggers:
  - Usage limits hit
  - Collaboration needs
  - Advanced features needed
- In-app upgrade prompts

Step 4: Sales Handoff (for Enterprise)
- High usage signals ‚Üí SDR outreach
- Multi-user teams ‚Üí Account expansion
- Security/compliance needs ‚Üí Enterprise sales
```

### Sales-Led (Enterprise) Playbook

**Framework (Salesforce/Oracle Model):**
```
Phase 1: Account Selection (ABM)
- ICP scoring (fit + intent)
- Buying committee identification
- Account research & planning

Phase 2: Outbound Orchestration
- Multi-channel sequences (email, phone, LinkedIn)
- Personalized messaging (account-specific value prop)
- Executive sponsorship (C-level to C-level)

Phase 3: Discovery & Demo
- MEDDIC qualification
- Custom demo (their use case)
- Technical deep-dive (SE/SA)

Phase 4: Evaluation & POC
- Proof of concept (30-60 days)
- Champion enablement (internal selling)
- Business case development (ROI)

Phase 5: Negotiation & Close
- Pricing proposal
- Security/legal review
- Executive close (if needed)
- Contract negotiation
```

## Industry-Specific GTM Strategies

### B2B SaaS (Horizontal Platform)
**Model: Slack, Zoom, Notion**
```
Motion: Bottom-up, Team ‚Üí Enterprise
Acquisition: PLG (freemium) + Sales (expansion)
Timeline: 3-6 months avg sales cycle
Metrics: Viral coefficient, team adoption, seats expansion
```

### Enterprise Software (Vertical Solution)
**Model: Salesforce, Oracle, SAP**
```
Motion: Top-down, Executive-led
Acquisition: Sales-led (ABM) + Partner channel
Timeline: 6-18 months sales cycle
Metrics: Deal size, win rate, implementation success
```

### Developer Tools (API-First)
**Model: Stripe, Twilio, AWS**
```
Motion: Developer ‚Üí Manager ‚Üí Executive
Acquisition: PLG (docs/APIs) + Usage-based revenue
Timeline: 1-3 months to paid conversion
Metrics: API calls, usage growth, developer NPS
```

### Infrastructure/Platform (Cloud)
**Model: AWS, Azure, GCP**
```
Motion: Multi-product land-and-expand
Acquisition: PLG (free tier) + Enterprise sales + Partners
Timeline: Varies (instant trial ‚Üí multi-year enterprise)
Metrics: Consumption revenue, workload migrations, ecosystem growth
```

```

### Microsoft Build Launch Approach
```
Developer-First:
- Technical preview (GitHub)
- Documentation before announcement
- Sample apps & templates
- Integration with existing tools

Enterprise Comfort:
- Security & compliance day-1
- Enterprise SLAs
- Premier support options
- Training & certification

Ecosystem Activation:
- Partner integrations (500+ at launch)
- ISV co-marketing
- System integrator enablement
```

### Stripe Product Philosophy
```
Principles:
1. "It just works" - Obsessive QA
2. Documentation = Marketing (best docs in industry)
3. Developer experience > everything
4. Transparent pricing (no hidden fees)
5. Quick time-to-value (7 minutes to accept payments)

Launch Tactics:
- Private beta (invite-only, 1000s of devs)
- Public beta (open, gather feedback)
- General availability (polished, documented)
- Continuous iteration (weekly deploys)
```

### Netflix Consumer Launch Tactics
```
Data-Driven:
- A/B test everything (artwork, messaging, features)
- Personalization from day 1
- Multi-variant testing (not just A/B)

Global Mindset:
- Localization (language, content, payment)
- Regional marketing (local influencers, culture fit)
- Staggered rollout (learn from each market)

Content Marketing:
- Show don't tell (video > text)
- User-generated content (social sharing)
- Influencer partnerships (authentic voices)
```
