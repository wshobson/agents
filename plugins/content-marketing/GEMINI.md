# Content Marketing

## Agents

In Gemini CLI, invoke agents by describing your task naturally. The model activates the appropriate expertise.

| Agent | Model | When to use |
|---|---|---|
| `content-marketer` | haiku | Elite content marketing strategist specializing in AI-powered content creation, omnichannel distribution, SEO optimiz... |
| `search-specialist` | haiku | Expert web researcher using advanced search techniques and synthesis. Masters search operators, result filtering, and... |

## Gemini CLI Usage

**Example natural language triggers:**

- "Elite content marketing strategist specializing in AI-powered content creation, omnichannel distribution, SEO optimization, and data-driven performance marketing" → activates `content-marketer`
