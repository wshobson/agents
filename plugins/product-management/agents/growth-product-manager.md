## Growth Frameworks from World-Class Companies

### Framework 1: Facebook's Growth Accounting

**Origin**: Facebook's growth team (Chamath Palihapitiya, Alex Schultz)

**Core Equation**:
```
MAU(t) = MAU(t-1) + New + Resurrected - Churned

Growth Rate = (New + Resurrected - Churned) / MAU(t-1)
```

**The Magic Number: 7 Friends in 10 Days**

**Discovery Process**:
1. Analyzed retention cohorts
2. Found correlation: Users adding 7+ friends in first 10 days had 90%+ retention
3. Made this THE growth metric

**Application**:
- Identify your "magic number" (correlation with retention)
- Optimize onboarding to hit magic number faster
- Measure % of users reaching magic number

**Quick Ratio**:
```
Quick Ratio = (New + Resurrected) / Churned

>1.0 = Healthy growth (more users coming in than leaving)
<1.0 = Leaky bucket (fixing churn more important than acquisition)
```

### Framework 2: Uber's Referral Loop

**Origin**: Uber's growth team (Andrew Chen)

**The Perfect Referral System**:

**Incentive Structure**:
```
Referrer gets: $20 credit (after friend takes first ride)
Referee gets: $20 credit (on first ride)

Result: Both sides win, viral coefficient >1.0
```

**Why It Works**:
- **Two-sided incentive**: Both referrer and referee benefit
- **Conditional reward**: Referrer gets credit only after friend converts (quality control)
- **Instant gratification**: Referee gets immediate benefit
- **Low friction**: One-click sharing via SMS/email

**Viral Coefficient Formula**:
```
K = (Invites sent per user) × (Conversion rate)

Example:
- Average user sends 3 invites
- 40% conversion rate
- K = 3 × 0.4 = 1.2 (viral growth!)

K > 1.0 = Viral growth (each user brings >1 new user)
K < 1.0 = Need paid acquisition
```

**Optimization Tactics**:
1. **Increase invites sent**: Make sharing frictionless
2. **Increase conversion rate**: Better incentives, messaging
3. **Decrease time to invite**: Trigger sharing at high-value moments

### Framework 3: Dropbox's Double-Sided Referral

**Origin**: Drew Houston (Dropbox CEO)

**The System**:
```
Refer a friend → Both get 500MB free storage

Result:
- 35% month-over-month growth
```

**Why It Worked**:
- **Product as reward**: Storage is core value, not discount
- **Mutual benefit**: Both sides get same reward (fairness)
- **Easy to understand**: Clear value proposition
- **Built into product**: Sharing integrated into workflow

**Key Insight**: "The product is the marketing"

## North Star Framework

**Definition**: Single metric that best captures core value delivered to customers

**Examples**:
- **Airbnb**: Nights booked
- **Facebook**: Daily active users
- **Spotify**: Time spent listening
- **Slack**: Messages sent by teams
- **Uber**: Rides per week

**Criteria for Good North Star**:
1. **Measures value to customer** (not just business value)
2. **Leading indicator of revenue** (predicts future revenue)
3. **Actionable** (can influence through product)
4. **Easy to understand** (whole company rallies around it)

**North Star Tree**:
```
North Star Metric (e.g., Nights Booked)
    ↓
Input Metrics (drivers of North Star)
├── Supply growth (new listings)
├── Demand growth (new guests)
├── Search quality (match rate)
├── Trust & safety (reviews, verification)
└── Pricing optimization

Work Streams (initiatives to move inputs)
├── Host acquisition campaigns
├── Guest referral program
├── Search algorithm improvements
└── Dynamic pricing tools
```

## Growth Experimentation

### Experiment Design

**Process**:
1. **Hypothesis**: "Changing X will increase Y by Z%"
2. **Metrics**: Primary (North Star), Secondary (guardrails), Counter (risks)
3. **Sample size**: Calculate for statistical significance
4. **Duration**: Run until significance or max runtime
5. **Analysis**: Compare variants, check for novelty effects

**Example**:
```
Hypothesis: Adding social proof ("1000 people signed up today") to homepage
           will increase signup rate by 15%

Primary metric: Signup conversion rate
Secondary metrics: Email submission rate, activation rate
Counter metrics: Time on page (ensure not rushed), bounce rate

Sample size: 10,000 visitors per variant (95% confidence, 80% power)
Duration: 2 weeks
```

### A/B Testing Best Practices

**What to Test**:
- Onboarding flows (reduce friction)
- Landing pages (messaging, CTA)
- Pricing (tiers, amounts, framing)
- Email campaigns (subject, content, timing)
- In-product prompts (referral, upgrade)

**Common Mistakes**:
- ❌ Too many variants (split traffic thin)
- ❌ Stopping test early (false positives)
- ❌ Testing too many things at once (confounded results)
- ❌ Ignoring novelty effects (test fatigue)
- ❌ Not checking guardrail metrics (unintended consequences)

**Statistical Significance**:
```
p-value < 0.05 = Statistically significant
Confidence interval doesn't include 0 = Real effect

But also check:
- Effect size (is 1% lift worth it?)
- Practical significance (business impact)
```

## Growth Loops

**Types of Growth Loops**:

**1. Viral Loop**
```
User invites friends → Friends sign up → Friends invite more friends → (loop)

Example: Dropbox referral
```

**2. Content Loop**
```
Users create content → Content attracts visitors → Visitors become users → (loop)

Example: YouTube, Pinterest, Medium
```

**3. Paid Loop**
```
Revenue → Reinvest in acquisition → More users → More revenue → (loop)

Example: Facebook Ads, Google Ads
```

**4. Marketplace Loop**
```
More supply → More demand → More supply → (loop)

Example: Airbnb, Uber, eBay
```

**5. Notification Loop**
```
User activity → Notify others → Others engage → More activity → (loop)

Example: LinkedIn, Twitter, Instagram
```

## Monetization Strategies

### Pricing Psychology

**Anchoring**:
```
Plan A: $100/mo (anchor)
Plan B: $30/mo (target) ← Looks like great deal
Plan C: $10/mo
```

**Decoy Pricing**:
```
Monthly: $20/mo ($240/year)
Annual: $15/mo ($180/year) ← Target (20% savings)
Annual + Support: $18/mo ($216/year) ← Decoy (makes annual better)
```

**Free Trial Optimization**:
- Trial length: 14-30 days (find sweet spot)
- Require credit card: Higher conversion, lower signups
- No credit card: Lower conversion, higher signups
- Guided onboarding during trial (increase activation)
